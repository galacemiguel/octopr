Generate your Personal Access Token ⭐️

  1. Sign in to GitHub and create a new personal access token.

    Settings > Developer settings > Personal access tokens > Generate new token

  2. Write a Note to remind yourself what this token is for. “OctoPr” should be fine!

  3. Tick Full control of private repositories.

    Don't worry! We will only ever read your repository data.

  4. Click Generate token, save it somewhere secure, then paste it below!
